# Summarises Sleep Information from Fitbit Raw Data

Creates a dataset containing summarised sleep data such as mean time in
bed, sleep duration and time awake

## Usage

``` r
sleepSummarise(DataFrame)
```

## Arguments

- DataFrame:

  A dataframe containing Fitbit sleep data.

## Value

A summarised dataset with average time in bed, asleep awake as well as
number of nights with or without missing sleep data for each participant

## Examples

``` r
if (FALSE) { # \dontrun{
hypometrics::sleepSummarise(DataFrame)
} # }
```
